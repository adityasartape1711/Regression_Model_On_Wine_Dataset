# Wine Price Prediction using Machine Learning

## Project Overview
This project predicts wine prices using advanced regression models based on wine quality, origin, and characteristics.

## Models Used
- Random Forest Regressor
- XGBoost Regressor (Best)
- LightGBM Regressor

## Techniques Applied
- Missing value handling
- Frequency encoding
- Outlier removal
- Log transformation
- Hyperparameter tuning (GridSearchCV)
- SHAP explainability

## Evaluation Metrics
- R² Score
- RMSE
- MAE

## Deployment
- Flask REST API
- FastAPI REST API
